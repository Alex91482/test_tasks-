Задание. Напишите модуль для Node.JS, который преобразует массив чисел в строку по примеру:
[1,2,3,4,5,6,7,8] -> ”1-8”
[1,3,4,5,6,7,8] -> ”1,3-8”
[1,3,4,5,6,7,8,10,11,12] -> “1,3-8,10-12”
[1,2,3] -> “1-3”
[1,2] -> “1,2”
[1,2,4] -> “1,2,4”
[1,2,4,5,6] -> “1,2,4-6”
[1,2,3,7,8,9,15,17,19,20,21] -> “1-3,7-9,15,17,19-21”
[1,2,3,4,5,6,100,1091,1999,2000,2001,2002] -> “1-6,100,1091,1999-2002”
[1] -> “1”
[1,3,5,7,9,11] -> “1,3,5,7,9,11”

Числа в массиве всегда целые, положительные, отсортированы по возрастанию.

Запустите “start_test.html” если браузер поддерживает инструменты разработчика, то в консоли выведется дополнительная информация.
Если хотите проверить работу скрипта то вызывайте функцию getStringArr() и передайте в нее массив с аргументами. 

В модуле реализовано:
1)	Проверка аргумента (Это массив чисел или массив с массивами).
2)	В зависимости от аргумента начнется либо преобразование массива, либо перебор массива с преобразованием массивов в строки
	(переборы массивов в таком случае будет запущены параллельно).
3)	Преобразования массива в строку проходят по следующей логике:
	- Создается два массива возвращаемый и промежуточный. Промежуточный массив используется для временного хранения значений. 
	Возвращаемый массив - это массив который будет преобразован в строку и отправлен как результат выполнения функции.
	- Если числа идут последовательно, то есть последнее выбранное число на единицу больше следующего, тогда число записывается в
	промежуточный массив.
	- Если число не последовательное (и не последнее) тогда проверяется промежуточный массив, если там есть значения то берется
	первое и последнее число из промежуточного массива (между ними ставится тире) и они записываются в возвращаемый массив, 
	"не последовательное" число добавляется в промежуточный массив. 
	- Если число последнее, то проверяется последовательное оно или нет. Если число последовательное, то число добавляется
	в промежуточный массив. Далее в возвращаемый массив добавляется, из промежуточного массива, первое и последнее число со
	знаком тире между ними. Если число не последовательное, то сначала из промежуточного массива извлекаются первое и последнее
	число (меду ними ставится тире) они добавляются в возвращаемый массив и после этого в возвращаемый массив добавляется 
	последнее число. 
4)	Возвращаемый массив преобразуется в строку и возвращается как результат выполнения функции. Если это массив со строками,
	то функция дожидается завершения выполнения всех Promise и только после этого возвращает массив со строками (при том 
	условии что все Promise завершились без ошибок).
